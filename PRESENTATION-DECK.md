## 03 / How It Works

### From data to decision in three layers.

#### 1. Live Data Ingestion
Bayan connects to your existing Microsoft Project Server via REST API. All project data — tasks, schedules, resources, progress — flows in real-time. No migration, no new tools.

#### 2. AI Intelligence Layer
GPT-5.2 analyzes the portfolio data combined with risk assessments, PM scores, and strategic objectives. It detects patterns, predicts outcomes, and runs Monte Carlo simulations.

#### 3. Action Execution
Bayan doesn't just recommend — it proposes specific actions (reassign resources, update progress). You confirm with one click. The change is written back to Project Server instantly.

> **Human-in-the-Loop Governance:** AI recommends, humans decide. Every action requires explicit confirmation.
